# Source inputs

This directory stores raw imported source artifacts.

Files in `source-inputs` are **not** accepted manifests and must not be treated as tested modlist outputs.

- `modlist.txt` is a Mod Organizer-style export used as source evidence for deterministic metadata conversion.
- Rows may be enabled, disabled, unmanaged, DLC, local, or Nexus-backed entries.
- Compatibility claims and manifest promotion require later human review.
